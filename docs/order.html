命令列表

在geosings.ui.MsgkeyWord.py中可以找到所有命令定义。这些命令可以通过主界面的输入区域输入，然后执行相应操作。

请注意，这些命令可以被修改， 可以按造你自己的习惯来配置。配置方法察看这里。

主界面命令列表：

命令名 命令用途
:o 打开一个图层的命令
:q 退出命令
:r 移除一个图层的命令
:top 使一个图层处于最高层
:v 使一个图层可见
:v! 使一个图层不可见
:h 显示帮助中心
:i 获取指定图层的图层信息
:tab 获取图层的属性表
:e 打开一个工程配置文件
:save 保存一个工程配置文件
:lab 添加一个图层的标注
:lab! 取消一个图层的标注
:exp 导出当前窗口中显示的地图到一个栅格数据格式(当前只支持GTiff)
:expall 导出当前窗口分辨率的整个地图到一个栅格数据格式(当前只支持GTiff)
:select 按照SQL语句条件选择
:search 搜索关键字

下面是主要命令的具体解释：

:o [path]

打开一个数据集

参数：

path 要打开的数据集的完整路径,根据可以打开数据集的路径表示,有可能不是传统表达形式路径，而是某种特定表达式（比如数据库连接字符串或者特殊数据集打开字 符串）

:q

退出主界面

参数：

无

:r layername

移除指定数据集

参数：

layername 要移除的数据集的名称

:top layername

把指定数据集移动到最上层

参数：

layername 要移除的图层的名称

:v layername

让指定数据集可见

参数：

layername 要可视的图层的名称

:v! layername

让指定数据集不可见

参数：

layername 要不可视的图层的名称

:h

调出帮助

参数： 无

:i layername

获取指定数据集的描述信息

参数：

layername 要获取信息的图层名称

:tab layername

获取指定数据集的属性表

参数：

layername 要获取属性表的图层名称

:e [path]

打开一个工程配置文件

参数：

path 要打开的配置文件的路径。

:save [path]

保存一个工程配置文件

参数：

path 要保存的配置文件的路径。

:lab layername [opt]

添加一个图层的标注

参数：

layername 要添加标注的图层。
opt 可选的一些参数，用“;”分割开，这些参数描述标签的属性。
下面是几个属性的解释和取值
选项 描述 数值类型 取值范例
field 要标注的字段名 字符串 name(默认)
font 字体名称 字符串 宋体(默认)
size 标注字体大小 整数 16(默认)
color 标注字体颜色 html形式的颜色 #ff66aa
bcolor 标注背景颜色 html形式的颜色 #ff0000
bold 是否粗体 0或者1 0

比如可以这样来设置：
field=xname;font=隶书;size=20;bcolor=#ff0000;color=#0000ff
这样就设置了标注字段为xname，字体为隶书，字体大小是20，背景色为红色，字体为蓝色的属性。

当然，后面应该还会添加其他属性，以增强功能。

:lab! layername

取消一个图层的标注

参数：

layername 要移除标注的图层。

:exp outputRasterPath

导出当前屏幕中的地图

参数：

outputRasterPath 要导出的图像文件名(包括路径)。

:expall outputRasterPath

导出当前绘制区域分辨率的整张地图

参数：

outputRasterPath 要导出的图像文件名(包括路径)。

:select sql

按sql语句选择

参数：

sql 要执行的选择语句。

:search sth

搜索某个关键字

参数：

sth 要搜索的关键字。
