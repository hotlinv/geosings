ReportLayerInfo是一个图层统计报告工具。该工具创建一个对于矢量或栅格数据的一个总体报告(html格式)。由linux_23(李林)编写。

实用类：geosings.tools.ReportLayerInfo.ReportLayerInfoCtrl

UI控制面板类：geosings.tools.ui.ReportLayerInfoUI.ReportLayerInfoUI

运行主界面因为没有控制参数所以略过。

主界面分为四个部分。一个是图层输入面板，一个是控制参数面板，一个是图层输出面板。一个是过程输出显示面板。

图层输入面板和图层输出面板的介绍可以看链接文档。

控制面板参数

暂时没有设置任何控制参数。
