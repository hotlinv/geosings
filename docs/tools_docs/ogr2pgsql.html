ogr2mysql是一个导入ogr矢量数据集到MySQL的工具。直接使用gdal提供的ogr2ogr实用程序。仅仅算是提供了一个GUI工具。

实用类：geosings.tools.Ogr2ogr.Ogr2ogr4intoPG

UI控制面板类：geosings.tools.ui.Ogr2ogr4intoPGUI.Ogr2ogr4intoPGUI
