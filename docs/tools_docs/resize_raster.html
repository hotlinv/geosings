RasterResizer是一个把栅格数据进行缩放的工具。该工具由linux_23(李林)编写。。

实用类：geosings.tools.RasterOperator.ResizeRaster

UI控制面板类：geosings.tools.ui.RasterResizerUI.RasterResizerUI

运行主界面如下：

主界面分为四个部分。一个是图层输入面板，一个是控制参数面板，一个是图层输出面板。一个是过程输出显示面板。

图层输入面板和图层输出面板的介绍可以看链接文档。最主要的核心在于控制 面板。

控制面板参数

缩放倍率
 缩放的倍率
默认为0.5(1.0/2)，注意，这里的倍率可以是可以计算的表达式，不一定都是浮点类型数字。甚至可以用raster.这样的表达来引用输入图像的属性(gdal.RasterDataset)。

输出

请注意：我们这里暂时只支持输出gtif的格式，所以输出文件名最好以tif为扩展名。
