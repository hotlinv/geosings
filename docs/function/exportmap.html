导出地图

返回主索引

九形提供导出地图的功能。有两种导出效果，一种是导出当前屏幕上的地图效果到一张图像上(使用的是:exp命令)，一种是导出当前屏幕分辨率的整张地图效果到一张图像上（使用的是:expall命令）。导出的地图图像格式是GTiff，暂时仅支持该格式。

导出当前屏幕分辨率的地图，就是把当前窗口绘制的图像导出成一张GTiff的图像，并且带地理信息。现在屏幕上画着什么，导出的图片就是什么样，没有变化，因为用的是同一套绘图程序和逻辑。导出命令像这样：

:exp e:/gisdata/gtif/screen.tif

导出当前屏幕分辨率的整张地图，就是把整个地图范围按照当前窗口分辨率绘制导出成一张GTiff的图像，并且带地理信息。注意，现在屏幕上画着什么，导出的图片可能会有变化，因为标注可能是按照整张图像范围来排列的，所以建议不要在配置了标注后再导出，可能不是你要的效果。当然除了标注位置外，其他的变化还是不大的。导出命令像这样：

:expall e:/gisdata/gtif/all.tif

因为导出的是整张地图，是从矢量到栅格数据，矢量是无限放大的，而栅格却是随着放大数据量呈级数增长的，所以如果你放矢量图到一个bt的程度，再用expall导出整张地图，恐怕你的硬盘会吃不消。
